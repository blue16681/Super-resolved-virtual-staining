# BCI 测试集评估使用说明（新增脚本）

## 1. 适用范围
本说明覆盖当前仓库新增/改造的测试与指标流程，面向 `dataset/BCIdataset/test/HE` 与 `dataset/BCIdataset/test/IHC` 的 PNG 配对数据。

## 2. 新增与改造内容
- `test_bci_png.py`
  - 新增。
  - 作用：直接读取 PNG 测试集进行推理，不再依赖 `.mat` 数据格式。
- `metrics_bci.py`
  - 新增。
  - 作用：对推理输出目录计算 `PSNR/SSIM/LPIPS`，并生成 CSV。
- `analysis_metrics.py`
  - 已改造为参数化 CLI。
  - 作用：支持单目录配对评估、双目录评估，并保留 `--legacy_windows_paths` 兼容旧流程。

## 3. 环境准备
建议先进入你训练模型时的 Python 环境（例如你自己的 `BBPM` 环境），然后再执行下面所有 `python` 命令。

### 3.1 确认 Python 可用
```bash
python --version
```

### 3.2 安装评估依赖（如果缺失）
```bash
python -m pip install scikit-image lpips
```

说明：
- `scikit-image` 用于 `PSNR/SSIM`。
- `lpips` 用于感知指标；如果不需要 LPIPS，可在指标脚本里关闭。

## 4. 推理流程（PNG 直读）

### 4.1 全量测试（skip 采样，推荐）
```bash
python test_bci_png.py \
  --model_path models/BBDM-20260426-1925/ema_0.9999_076000_compress.pt \
  --lr_data_dir dataset/BCIdataset/test/HE \
  --hr_data_dir dataset/BCIdataset/test/IHC \
  --batch_size 4 \
  --num_samples 977 \
  --cond_channels 3 \
  --out_channels 3 \
  --sampler skip \
  --avg_steps 50 \
  --large_size 256 \
  --crop_mode center \
  --save_dir outputs_bci_test_png_skip50
```

### 4.2 快速冒烟测试（子集）
```bash
python test_bci_png.py \
  --model_path models/BBDM-20260426-1925/ema_0.9999_076000_compress.pt \
  --lr_data_dir dataset/BCIdataset/test/HE \
  --hr_data_dir dataset/BCIdataset/test/IHC \
  --batch_size 1 \
  --num_samples 64 \
  --cond_channels 3 \
  --out_channels 3 \
  --sampler skip \
  --avg_steps 50 \
  --large_size 256 \
  --crop_mode center \
  --save_dir outputs_bci_test_png_skip50_n64
```

### 4.3 推理输出结构
在 `--save_dir` 下生成：
- `xxx.png`：模型预测结果
- `xxx_target.png`：GT（用于评估）
- `xxx_inp.png`：条件输入可视化

## 5. 指标计算流程

你现在有两种指标入口，二选一即可。

### 5.1 使用改造后的 `analysis_metrics.py`（推荐）

#### 同目录配对模式（`xxx.png` 对 `xxx_target.png`）
```bash
python analysis_metrics.py \
  --pred_dir output/outputs_bci_test_png_skip50 \
  --model_name bci_skip50
```

#### 不计算 LPIPS（加速）
```bash
python analysis_metrics.py \
  --pred_dir output/outputs_bci_test_png_skip50 \
  --model_name bci_skip50_nolpips \
  --disable_lpips
```

#### 双目录配对模式（同名文件匹配）
```bash
python analysis_metrics.py \
  --pred_dir /path/to/preds \
  --gt_dir /path/to/gts \
  --model_name bci_two_dir_eval
```

### 5.2 使用新增 `metrics_bci.py`
```bash
python metrics_bci.py \
  --pred_dir outputs_bci_test_png_skip50 \
  --csv_name metrics.csv
```

可选：
```bash
python metrics_bci.py \
  --pred_dir outputs_bci_test_png_skip50 \
  --disable_lpips
```

## 7. 常见问题与处理

### 7.1 `ModuleNotFoundError: No module named 'numpy'`
当前 shell Python 不是项目环境。请改用：
```bash
python ...
```

### 7.2 `No module named 'skimage'`
安装依赖：
```bash
python -m pip install scikit-image
```

### 7.3 `LPIPS mean: N/A`
说明 `lpips` 未安装或初始化失败。可执行：
```bash
python -m pip install lpips
```

### 7.4 推理速度慢
- 先用 `--num_samples 64` 做冒烟测试。
- 再跑全量 `977`。
- 可尝试增大 `--batch_size`（先监控显存是否溢出）。

### 7.5 图像尺寸报错（crop 相关）
- 当前脚本默认按 `--large_size` 裁剪。
- 若测试图尺寸小于 `large_size`，请降低 `--large_size` 或更换输入。

## 8. 参数速查（`test_bci_png.py`）
- `--model_path`：建议传 `*_compress.pt`，脚本会自动推断对应主模型 `.pt`。
- `--sampler`：`skip` / `mean` / `standard`。
- `--avg_steps`：`skip` 或 `mean` 采样时使用。
- `--num_samples`：`<=0` 表示全量。
- `--crop_mode`：`center` 或 `top_left`。

## 9. 参数速查（`analysis_metrics.py`）
- `--pred_dir`：预测输出目录。
- `--model_name`：输出 CSV 前缀。
- `--disable_lpips`：关闭 LPIPS。
- `--gt_dir`：可选，双目录评估。
- `--legacy_windows_paths`：启用旧硬编码路径逻辑。
